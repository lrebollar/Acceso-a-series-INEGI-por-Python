# Acceso-a-series-INEGI-por-Python
Recientemente INEGI cambió muchas de las claves de las series de tiempo para el manejo de la información estadística. Anteriormente he utilizado librerias como INEGIpy para importar estas pero con estos cambios ha dejado de funcionar, por lo que me he dado a la tarea de crear un pequeño código para acceder a esta información nuevamente
